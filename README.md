# SUPER-STORE-SALES-DASHBOARD
Overview

The Super Store Sales Dashboard is an interactive Power BI project designed to analyze sales performance, profit trends, customer segments, shipping modes, and regional sales distribution. The dashboard provides actionable insights through dynamic visualizations, KPIs, filters, and forecasting features to support business decision-making.
