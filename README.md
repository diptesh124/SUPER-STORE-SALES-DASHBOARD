# SUPER-STORE-SALES-DASHBOARD
# 📊 Super Store Sales Dashboard & Sales Forecasting

An interactive **Power BI Dashboard** built using the Super Store Sales dataset to analyze sales performance, profit trends, customer behavior, and future sales forecasting. The dashboard helps businesses make data-driven decisions through dynamic visualizations and predictive analytics.

---

## 🚀 Project Overview

This project provides a complete business intelligence solution for analyzing retail sales data. It includes KPI tracking, regional performance analysis, customer segmentation, product insights, and sales forecasting.

### Key Objectives
- Monitor overall business performance
- Analyze sales and profit trends
- Identify top-performing regions and products
- Understand customer purchasing behavior
- Forecast future sales using historical data

---

## 📌 Dashboard Features

### 📈 Sales Analysis Dashboard
- Total Sales KPI
- Total Orders KPI
- Total Profit KPI
- Average Delivery KPI
- Monthly Sales Trend
- Monthly Profit Trend
- Sales by Segment
- Sales by Payment Mode
- Sales by Category
- Sales by Sub-Category
- Sales by Ship Mode
- State-wise Sales & Profit Analysis
- Region Filters (Central, East, South, West)

### 🔮 Sales Forecast Dashboard
- Historical Sales Trend Analysis
- Time-Series Forecasting
- State-wise Sales Comparison
- Interactive Date Filters
- Future Sales Prediction
